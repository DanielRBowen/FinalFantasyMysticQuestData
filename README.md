## What’s in this repo
- `FFMQ_Complete_LLM_Guide.markdown` – unified, LLM-friendly guide compiled from all sources (primary artifact).
- Source CSV/TXT data files used to build the guide (armor, items, spells, locations, bestiary, etc.).
